# Security Policy

## 敏感信息

请不要在 issue、pull request、截图或代码中提交网易云音乐 Cookie、账号、手机号、邮箱或其他身份凭据。

本项目支持通过 `NETEASE_COOKIE` 环境变量或本地 `bridge.config.json` 配置 Cookie。`bridge.config.json` 默认被 `.gitignore` 忽略，只应保存在本机。

## 报告安全问题

如果你发现会泄露 Cookie、越权读取本地文件、或把本地凭据发送到非网易云域名的问题，请通过私下渠道联系维护者，避免在公开 issue 中贴出可复现的敏感数据。
